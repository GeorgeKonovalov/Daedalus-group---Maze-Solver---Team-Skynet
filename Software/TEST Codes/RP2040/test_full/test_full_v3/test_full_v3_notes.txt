Main changes in this version:

- Added separate IR Test and Ultrasonic Test screens.
- Expanded the execution screen to show mode, attempt, time, event, heading, ultrasonic readings, and IR readings.
- Added a pre-start Event Test area with test cases for Corridor, T-Junction, Dead-end, Start, Finish, Left Turn, Right Turn, and T-Straight.
- Added confirmation/start sequencing so trials can restart from the confirmation screen after Finish.
